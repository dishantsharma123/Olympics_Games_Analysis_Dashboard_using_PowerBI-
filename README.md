# 🏅 Olympic Games Analysis Dashboard

## 📌 Project Overview

This project provides an interactive analysis of Olympic Games data using Power BI. The dashboard explores athlete participation, medal distribution, country performance, gender trends, and historical Olympic insights.

The objective is to uncover meaningful patterns and trends from Olympic Games data and present them through visually appealing and interactive dashboards.

---

## 🎯 Business Objectives

- Analyze medal distribution across countries.
- Identify top-performing athletes and nations.
- Compare Summer and Winter Olympic performances.
- Explore gender participation trends over time.
- Track Olympic growth and participation history.
- Discover sports with the highest medal counts.

## 📈 Key Insights

### 🥇 Medal Leaders
- USA dominates overall medal count.
- Soviet Union and China consistently rank among top-performing nations.

### 🌍 Participation Growth
- Athlete participation has increased significantly over the decades.
- More countries are participating in recent Olympic Games.

### 👩‍🦰 Gender Diversity
- Female athlete participation has grown steadily.
- Recent Olympics show near-balanced representation.

### 🏃 Sports Performance
- Athletics and Swimming contribute the highest number of medals.
- Team sports attract the largest athlete participation.
